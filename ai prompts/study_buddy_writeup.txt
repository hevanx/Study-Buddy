● Here's a polished version:

Study Buddy is a web-based study tracking application that allows users to log study time, earn XP, advance through ranks (Unranked through Diamond), collect cosmetic rewards, and connect with friends. This milestone focused on finalizing deployment and ensuring the application runs reliably from a self-contained local environment.

The back-end is built with Node.js and Express, organized into dedicated route modules for authentication, study progress, the XP/timer system, friends, and cosmetics. Data is persisted in a MariaDB database that is automatically initialized with schema and sample data on first launch. Five pre-seeded user accounts spanning all rank tiers are included so the application can be explored immediately without manual setup.

The entire stack, Node.js API, MariaDB database, Nginx static file server, and phpMyAdmin, is containerized and orchestrated with Docker Compose. Running a single docker-compose up command from the project directory builds and starts all services with no additional configuration required. The front end is served by Nginx on port 8081, the API runs on port 3000, and phpMyAdmin is available on port 8082 for database inspection. The database volume is persistent across restarts and can be fully reset via docker-compose down -v.

All known bugs identified during testing were resolved prior to this submission. The application was verified to register and log in users, track and persist study sessions, update XP and rank correctly, and load friend and leaderboard data as expected.
